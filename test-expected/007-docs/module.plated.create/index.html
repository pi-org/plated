module.plated.create

The plated module only exposes one function, which is used to create and bind state data which then provides the rest of the function.

plated=module.plated.create(opts,plated)

opts is an object of options and plated is an optional input if provided it will be modified else a new object will be created, either way it will be returned by this function.

We also load and setup this and all the builtin plugins so after calling this we are good to go.
